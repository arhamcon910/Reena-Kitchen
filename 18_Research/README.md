# 18_Research

Competitive analysis and user research.

## Key Findings

1. Indian vegetarian households make ~21 cooking decisions per week
2. Festival calendar drives 30–40% of meal variety changes
3. Pantry staples (onion, tomato, dal, ghee) appear in 85%+ of meals
4. Health goals primary driver in 40% of households
5. Cuisine variety is a strong dissatisfier after 3+ same-cuisine meals
