## Mission

Build the world's most intelligent, privacy-first AI kitchen and household operating system for Indian vegetarian families — delivering personalised meal recommendations, household task orchestration, and family intelligence that improves with every interaction.

## Scope

### Phase 1 — AI Kitchen Core (RC1 ✅)
7-engine decision system · 48 vegetarian recipes · Family memory · AI chat · Meal planner · Pantry · 847 tests · 75/75 audit

### Phase 2 — Household Intelligence (v1.1 📋)
Sentry · Server sync · Shopping · Finance · Push notifications

### Phase 3 — Enterprise OS (v2.0 📋)
Laundry · Cleaning · Maintenance · Garden · Pets · Hindi localisation

## Architecture Principles

1. **Privacy-First** — device-local by default, server sync opt-in
2. **Offline-First** — all core features work without network
3. **Vegetarian-Always** — 9-layer enforcement, never configurable
4. **Learning-Progressive** — smarter with every family interaction
5. **Explainable** — every recommendation carries a human-readable reason
6. **Versioned-Always** — all SP schemas versioned for non-destructive migration
7. **Test-Gated** — no code reaches distribution without passing gate suite
