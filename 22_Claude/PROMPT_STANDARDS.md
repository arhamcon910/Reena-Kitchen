# Prompt Standards

## Template

```
Task: [Clear task name]
Goal: [One sentence]
Requirements:
- [Specific bullet requirements]
- Reference existing class names and file paths
- State vegetarian policy if food-related
Deliver:
1. Files modified
2. [Specific deliverables]
Gate: [What must be true before complete]
/oodo
```

## Rules

1. Always end with `/oodo`
2. Reference existing architecture explicitly
3. State vegetarian policy requirement for all food/AI features
4. Include gate criteria
5. Specify test assertion count target
