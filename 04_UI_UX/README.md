# 04_UI_UX

Design system and UX documentation.

## Design System

Implemented in `09_Packages/design_system/`.

- **Base grid:** 4pt
- **Spacing:** xs=4, sm=8, md=16, lg=24, xl=32, xxl=48
- **Corner radius:** sm=8, md=12, lg=16, xl=24
- **Text scaling:** `textScaler.clamp(min: 0.8, max: 1.4)` on hero elements
